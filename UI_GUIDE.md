## üé® Theme Consistency Rules

1. **Backgrounds**
   - Page background: #282c34
   - Card/Paper: #21252b
   - Elevated elements: #2c313c

2. **Borders**
   - Default: #181a1f
   - Focus: #61afef

3. **Text**
   - Primary: #abb2bf
   - Secondary: #5c6370
   - Headings: #ffffff

4. **Interactions**
   - Hover: rgba(97, 175, 239, 0.1)
   - Selected: rgba(97, 175, 239, 0.2)
   - Active: #61afef

5. **Shadows**
   - Small: 0 2px 4px rgba(0, 0, 0, 0.2)
   - Medium: 0 4px 8px rgba(0, 0, 0, 0.3)
   - Large: 0 8px 16px rgba(0, 0, 0, 0.4)

6. **Border Radius**
   - Small: 4px (chips, small buttons)
   - Medium: 8px (inputs, buttons, cards)
   - Large: 12px (large cards)
